## 📝 VALIDAÇÃO DO BANCO

Execute esta query para verificar todos os 10 negócios:

```sql
SELECT 
  u.id,
  u.name,
  u.email,
  CASE 
    WHEN dc.id IS NOT NULL THEN 'Delivery'
    WHEN sc.id IS NOT NULL AND pc.id IS NOT NULL THEN 'Serviços + Produtos'
    WHEN sc.id IS NOT NULL THEN 'Serviços'
    WHEN pc.id IS NOT NULL THEN 'Produtos'
    ELSE 'Sem configuração'
  END as tipo_negocio,
  COALESCE(
    (SELECT COUNT(*) FROM menu_items WHERE user_id = u.id),
    0
  ) + COALESCE(
    (SELECT COUNT(*) FROM products WHERE user_id = u.id),
    0
  ) + COALESCE(
    (SELECT COUNT(*) FROM scheduling_services WHERE user_id = u.id),
    0
  ) as total_items,
  CASE 
    WHEN ai.is_active THEN 'IA Ativa'
    ELSE 'IA Inativa'
  END as status_ia
FROM users u
LEFT JOIN delivery_config dc ON u.id = dc.user_id
LEFT JOIN products_config pc ON u.id = pc.user_id
LEFT JOIN scheduling_config sc ON u.id = sc.user_id
LEFT JOIN ai_agent_config ai ON u.id = ai.user_id
WHERE u.email LIKE '%@flow.test'
ORDER BY u.created_at DESC;
```
